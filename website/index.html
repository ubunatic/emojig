emojig

all your emoji are belong to us — instant terminal emoji picker

$ curl -fsSL https://ubunatic.com/emojig/install.sh | sh

Features

  • 🏎️  Fast & tiny — 340 KB static binary, < 2 MB RAM, zero dependencies
  • 😀  1,870 emojis — embedded at compile time, fuzzy search with stem/plural fallback
  • 🔍  Inline TUI or floating window — works over SSH, in VT, anywhere terminals run
  • ⌨️  Shell widget Ctrl+E — picks emoji inline at your cursor, no workflow break
  • 🌗  Dark / light / system theme — auto-detected via OSC 11, Tab to toggle & save
  • 📋  Clipboard + stdout — copies via wl-copy / xclip and prints to stdout

In the editor too

Controls

Key Action
Type Fuzzy filter — stems & plurals work (cats → cat)
↑ ↓ ← → Navigate the grid
Enter or click Select & copy to clipboard
Tab Cycle theme (dark / light / system) and save
Esc / Ctrl+C Cancel

Install

Downloads the latest static binary for your architecture and sets up shell integration:

$ curl -fsSL https://ubunatic.com/emojig/install.sh | sh

Add one line to your shell rc and press Ctrl+E at any prompt. Full docs, source, and releases on Codeberg.

Requirements: Linux x86_64 or aarch64. GUI mode needs foot + Wayland or X11. Clipboard: wl-copy (Wayland) or xclip (X11).
