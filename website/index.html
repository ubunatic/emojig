Status Quo & Features

Optimized system-level mechanics, fully standalone, with absolute zero background daemons.

🚀

Zero-Allocation Performance

Engineered in pure Zig. Performs zero heap allocations during the active search loop. Delivers instantaneous renders under 2.0 MB resident memory.

🖥️

Dual TUI & GUI Modes

Fzf-like auto-detection. Launches inline in active terminals, or automatically spawns a borderless graphical popup window when executed from hotkeys.

🔍

Intelligent Fuzzy Match

Space-separated multi-term matching (AND semantics) with automatic plural (cars → car), verb stem (running → run), and trailing 'e' fallback corrections.

🌗

OSC 11 Theme Auto-Detect

Queries the host terminal background via standard OSC 11 sequences to choose light/dark modes. Toggle instantly via Tab and save to local config.

🛡️

Terminal State Safety

Custom panic handlers and native POSIX signal captures (`SIGINT`, `SIGTERM`) ensure terminal raw settings, mouse tracking, and cursors are fully restored on exit.

📦

Embedded Emoji Packer

Compile-time database packing serialized via Go pack tools directly into the compiled Zig binary. Zero runtime file/database disk reads during queries.

✨

Floating Desktop Integration

Spawns a beautifully compact floating window using standard native terminals.

Keyboard Controls

Input Action Description
Ctrl+E Launch picker at the shell prompt; inserts pick into the command line
Type Text Live fuzzy search (auto-stems & plurals)
↑ ↓ ← → Picker: 2D grid selection (wraps). Shell: ↑↓ history, ←→ move cursor
Enter / Click Confirm selection & copy to clipboard
Tab Cycle theme (dark / light / system) & save
Esc / Ctrl+C Cancel and restore terminal TTY raw settings

Setup & Requirements

  • ✓
    Supported OS: Linux on standard x86_64 or aarch64. Fully runs over SSH or direct VTs.
  • ✓
    GUI Float Terminals: Spawns via foot (preferred), kitty, alacritty, wezterm, ghostty, konsole, gnome-terminal, or ptyxis.
  • ✓
    Clipboard Tools: Copies to clipboard automatically using wl-copy (Wayland) or xclip (X11) if available.
  • ✓
    Shell Sourcing (Ctrl+E): Add source ~/.local/share/emojig/shell/emojig.<shell> to your .bashrc, .zshrc, or fish config to enable the trigger.
